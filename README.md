# Export Dashboard (Excel)

This project presents an Excel dashboard for analyzing export performance using real-world data.

## Features:
- KPI metrics (Total Export 2023–24, Total Export 2024–25, Growth %)
- Top 10 Export Countries (Bar Chart)
- Top 5 Countries Share (Pie Chart)
- Year-wise Comparison (2023–24 vs 2024–25)

## Tools Used:
- Microsoft Excel
